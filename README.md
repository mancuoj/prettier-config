No semi, single quotes, trailing comma.

## Usage

```shell
ni -D prettier @mancuoj/prettier-config
```

In your `package.json`:

```json
{
  "scripts": {
    "format": "prettier --cache --write ."
  },
  "prettier": "@mancuoj/prettier-config"
}
```
